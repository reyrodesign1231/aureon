# Aureon

Aureon is a cinematic WebGL case study about bronze, time, sculpture, and digital material.

The page uses a 3D horse sculpture, scroll-based camera movement, atmospheric lighting, sparks, and animated typography to create a museum-like luxury experience in the browser.

## Highlights

- Three.js / WebGL scene
- Scroll-driven 3D camera motion
- Cinematic bronze lighting
- Animated story sections
- Mobile-optimized navigation and model framing

## Tech Stack

HTML, CSS, JavaScript, Three.js, WebGL
